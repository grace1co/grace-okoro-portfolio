# Grace Okoro Portfolio

A personal portfolio website showcasing my software engineering projects, technical experience, certifications, and skills. The site highlights full-stack, AI-powered, and data-driven applications through interactive project sections, responsive design, and a clean visual layout.

## Live Site

https://grace-okoro-portfolio.vercel.app/

## Features

* Interactive project showcase with expanded project details
* Technical experience, certifications, and skills sections
* Responsive layout for desktop, tablet, and mobile screens
* Dark and light mode support
* Smooth animations and interactive UI elements
* Project icons, certification badges, and technology visuals
* Contact and GitHub links for recruiters and collaborators

## Built With

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* Radix UI
* Lucide Icons
* GSAP
* React Spring
